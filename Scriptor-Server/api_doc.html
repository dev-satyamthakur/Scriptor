API Documentation

1. Generate Article

Endpoint: /generate-article

Method: POST

Description: Handles requests for article generation based on a provided topic.

Request

Content-Type: application/json

Body:

  • topic: string - The topic for which the article is to be generated

Response

Status Code: 200 OK

{ "message": "Article generated", "article": "string" // The generated article content }

Status Code: 400 Bad Request

{ "error": "Topic is required" }

Example Request

curl -X POST http://:/generate-article \ -H "Content-Type: application/json" \ -d '{"topic": "Artificial Intelligence"}'

Example Response

{ "message": "Article generated", "article": "Artificial Intelligence (AI) is a field of computer science that aims to create machines that can perform tasks that typically require human intelligence..." }

2. Generate Article HTML

Endpoint: /generate-article-html

Method: POST

Description: Handles requests for generating HTML from the article.

Request

Content-Type: application/json

Body:

  • main_article: string - The main article content
  • number_of_sections: integer - The number of sections to divide the article into
  • words_per_section: integer - The number of words per section
  • image_urls: array of strings - An array of image URLs to include in the HTML

Response

Status Code: 200 OK

{ "message": "HTML generated", "html_output": "string" // The generated HTML content }

Status Code: 400 Bad Request

{ "error": "Missing required input parameters" }

Example Request

curl -X POST http://:/generate-article-html \ -H "Content-Type: application/json" \ -d '{ "main_article": "Artificial Intelligence is transforming the world...", "number_of_sections": 3, "words_per_section": 100, "image_urls": ["http://example.com/image1.jpg", "http://example.com/image2.jpg"] }'

Example Response

{ "message": "HTML generated", "html_output": "

Introduction

Artificial Intelligence is transforming the world...

Applications

AI is used in various fields...

" }
